You are building a complete, production-ready website for "NB Therapy", a Belgian female wellness practitioner offering Hijama, massage, Ruqya Nuranya, and therapeutic listening sessions. The site targets French-speaking Belgian women.

## DESIGN DIRECTION
Base the design on this approved template direction: soft pastel feminine aesthetic, light backgrounds, elegant serif typography (Cormorant Garamond for headings, Jost for body), rose/lavender/sage color palette. NO emojis anywhere — replace all icons with clean SVG illustrations or geometric ornamental shapes. Elevate the design quality significantly: think high-end French wellness brand (think Aesop, Sisley, or Clarins level of refinement). Use subtle CSS animations, generous whitespace, fine decorative lines, and layered depth with soft shadows.

## TECH STACK
- Next.js 14 (App Router)
- Tailwind CSS (custom palette, no default colors)
- Framer Motion for animations
- Resend or Nodemailer for transactional emails
- Prisma + SQLite (or PostgreSQL) for appointment storage
- react-hook-form + zod for form validation
- date-fns for date/time handling
- Vercel-ready deployment config

### 2. `/api/book` — POST endpoint
On form submission:
1. Save appointment to database (status: PENDING)
2. Send confirmation email to client (HTML template, branded rose/cream design):
   - Subject: "Votre demande de rendez-vous — NB Therapy"
   - Body: confirm name, service, date/time, next steps
3. Send medical questionnaire email to client (send 5 minutes after booking, or immediately):
   - Subject: "Questionnaire médical — NB Therapy"
   - Fields to fill: pathologies existantes, médicaments en cours, allergies connues, grossesse en cours (oui/non), contre-indications connues, informations complémentaires
   - Design: clean HTML form-style email with a link to a hosted form page OR inline questions with a reply instruction
4. Send pre-session recommendations email (send 24h before appointment via cron job):
   - Subject: "Recommandations avant votre séance — NB Therapy"
   - Content: drink water, avoid heavy meals 2h before, wear comfortable clothing, arrive 5 min early, bring any relevant medical documents
5. Send reminder email 48h before appointment:
   - Subject: "Rappel de votre rendez-vous — NB Therapy"
   - Body: remind date, time, service, WhatsApp contact for changes

### 3. `/api/cron/reminders` — GET endpoint (Vercel Cron)
- Runs daily at 9h00
- Queries all appointments scheduled 48h from now (status: CONFIRMED)
- Sends reminder emails
- Queries all appointments scheduled 24h from now
- Sends pre-session recommendation emails
- Configure in vercel.json: `{ "crons": [{ "path": "/api/cron/reminders", "schedule": "0 9 * * *" }] }`

## DATABASE SCHEMA (Prisma)
```prisma
model Appointment {
  id          String   @id @default(cuid())
  firstName   String
  lastName    String
  phone       String
  email       String
  service     String
  date        DateTime
  message     String?
  status      String   @default("PENDING") // PENDING | CONFIRMED | CANCELLED
  token       String   @unique @default(cuid())
  questionnaireCompleted Boolean @default(false)
  reminderSent Boolean @default(false)
  recommendationSent Boolean @default(false)
  createdAt   DateTime @default(now())
}
```
